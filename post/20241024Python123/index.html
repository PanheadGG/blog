顺序结构简单编程1

将输入的分钟数转换为某某小时某某分钟。
运行样例：
输入

78

输出

1小时18分

展开/关闭答案

#include<iostream>
using namespace std;
int main() {
	int min;
	cin >> min;
	cout << min / 60 << "小时" << min % 60 << "分钟";
	return 0;
}

存款本利计算

设银行定期存款的年利率为3.15%，存款年限和存款本金由键盘输入，试编程计算到期后的本利之和。
假设不考虑税率。
利息（年）=本金×年利率（百分数）×存期
运行样例：
输入

15000 3

输出

total=16417.5

展开/关闭答案

#include<iostream>
using namespace std;
int main() {
	float money;
	int year;
	cin >> money >> year;
	cout << "total=" << money + money * 0.0315 * year;
	return 0;
}

求 a + | b |

输入

1.3 -5.6

输出

1.3 + | -5.6 | = 6.9

展开/关闭答案

#include<iostream>
#include<cmath>
using namespace std;

编写程序求华氏温度 F 对应的摄氏温度C

编写程序求华氏温度 F 对应的摄氏温度C
摄氏温度 C=(5/9)(F-32)
输入

100

输出

37.7778

展开/关闭答案

#include<stdio.h>
int main() {
	float f;
	scanf("%f", &f);
	printf("%.4f", 5.0 / 9.0 * (f - 32.0));
	return 0;
}

从键盘输入3个数, 输出最大数

输入

32 67 8

输出

max_number=67

展开/关闭答案

#include<iostream>
using namespace std;
int main() {
	int a, b, c, max;
	cin >> a >> b >> c;
	max = a;
	if (b > max) max = b;
	if (c > max) max = c;
	cout << "max_number=" << max;
	return 0;
}
